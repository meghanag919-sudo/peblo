# 🌟 Peblo – AI Story Buddy

An interactive AI-powered storytelling and quiz application built with React and Vite. Peblo creates engaging learning experiences by combining stories with quizzes, allowing users to improve comprehension and knowledge through interactive content.

---

## 📌 Overview

Peblo is designed to make learning enjoyable through story-based interactions. Users can read AI-generated stories and answer related quiz questions to test their understanding. The application features a clean, responsive interface and provides instant feedback for an engaging user experience.

---

* 📚 Interactive story experience

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/meghanag919-sudo/peblo.git
```

### Navigate to Project Folder

```bash
cd peblo
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

Open:

```
http://localhost:5173
```

## 🎯 Workflow

1. User reads a story.
2. A related quiz question is displayed.
3. User selects an answer.
4. The application validates the response.
5. Feedback is shown instantly.
